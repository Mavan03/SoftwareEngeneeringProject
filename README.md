#Trust Issues
**Trust Issues** is a 2D puzzle-platformer.
The core concept of the game is deception: not everything is what it seems. The player must navigate through levels where solid-looking floors might be fake, and invisible walls might block the path.
---
##Gameplay Features

* **Deceptive Mechanics:**
* **Fake Floors:** Tiles that look solid but allow you to fall through.
* **Invisible Walls:** Invisible tiles that block your movement.

* **Enemy AI:**
* **Walker:** Patrols platforms and turns around at edges.
* **Chaser (Bat):** Detects the player and flies towards them.
* **Trap (Spikes):** Stationary hazards.

* **Physics System:** Custom collision detection with gravity and momentum.
* **Level System:** Multiple levels with increasing difficulty and a Level Select screen.
---
## Controls
The game supports multiple input layouts (QWERTY/AZERTY):
| Action | Keys |
| --- | --- |
| **Move Left** | `A` / `Left Arrow` / `Q` |
| **Move Right** | `D` / `Right Arrow` |
| **Jump** | `Space` / `Up Arrow` / `W` / `Z` |
| **Menu Navigation** | Mouse Click |
